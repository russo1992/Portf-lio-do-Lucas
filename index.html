MINHAS ESPECIALIDADES

Testes Manuais

Realizo manualmente, ou seja,reproduzo as ações dos usuários para validar se determinada funcionalidade ou correção está funcionando conforme os usuários esperam. Existem diversas técnicas de testes manuais

Testes Automatizados

Uso programas que executam testes em softwares que estão em construção de uma forma padronizada, sem ser necessário a intervenção humana. Pois, tais testes possuem funcionalidades capazes de testar de forma automática todos os aspectos de uma plataforma, com o intuito de assegurar um desempenho adequado.

Git/Github

Git e Github são utilizados no dia a dia das pessoas que criam softwares por um motivo bem simples: ter uma forma fácil de gerenciar o código-fonte da aplicação, do sistema e do produto.
